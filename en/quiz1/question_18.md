Some Python code checks the speed of a car.

A variable called `speed` has the number `30` stored in it. 

What would be displayed when this code runs? 

--- code ---
---
language: python
---
speed = 30

if speed == 40:
  print('Fast')
elif speed == 50:
  print('Pretty quick')
elif speed == 60:
  print('Very fast')
else:
 print('Speed not recognised')
